Depression Detection Using Social Media

This project aims to develop an intelligent system that detects early signs of depression by analysing the language a person uses on social media — specifically the emotional weight of the positive and negative words they write. A machine learning classifier is trained on over 267,000 real-world labelled posts from public mental health communities, learning to distinguish concerning language from neutral text. The current implementation delivers a fully working web application with clinical screening, real-time text analysis, a supportive chatbot, and personal progress tracking. Future scope includes passive analysis directly from a user's social media feed.

Problem Statement

Digital text communication — especially social media — often reflects emotional and cognitive states through the type of language a person uses. Existing systems rely heavily on self-reporting and fail to detect subtle behavioural signals embedded in everyday writing. This project addresses the need for a privacy-preserving system that passively analyses the positive and negative language patterns in a person's social media text to estimate early depression risk in a non-intrusive manner.

Literature Review / Market Research

The study on touchscreen typing pattern analysis like "Touchscreen typing pattern analysis for remote detection of the depressive tendency" by Rafail-Evangelos Mastoras et al. demonstrates that depressive tendencies influence psychomotor behaviour and language patterns. Further NLP research (CLPsych shared tasks, EMNLP mental health studies) confirms that social media language reliably signals distress — users exhibit measurable shifts toward negative affect words, hopelessness expressions, and first-person singular language.

Research Gap / Innovation

Depression is often underreported and socially stigmatised, leading to delayed diagnosis. Most academic NLP systems remain offline research tools inaccessible to real users. This project bridges that gap — training on 267,782 posts across three real-world datasets and deploying as a live web application. Future scope extends to directly ingesting a user's own social media writing for passive, ongoing risk estimation without requiring manual input.

What's Implemented

Four fully working modules in the current web app. Social media feed integration is the planned next phase.

PHQ-9 Screening Quiz

Done

The clinically validated 9-question Patient Health Questionnaire. Users rate each symptom from 0–3. The app calculates a score, maps it to a severity level (Minimal / Mild / Moderate / Moderately Severe / Severe), and saves it for progress tracking.

ML Text Analysis

Done

The trained TF-IDF + Logistic Regression model analyses any text the user types and outputs a probability score, mapped to Low / Medium / High concern. Trained on 267,782 posts, 93.9% accuracy. This is the core module that will later connect to social media feeds.

Supportive Chatbot

Done

A keyword-driven conversational interface that detects emotional tone — sadness, anxiety, anger, or positive affect — and responds with contextually appropriate supportive replies. Designed to provide immediate, non-clinical emotional acknowledgment to the user.

User: I feel tired and hopeless

Bot: That sounds really heavy. I hear you. Would you like to talk more about what's been going on?

Progress Tracker

Done

Stores the last 5 PHQ-9 scores and displays session-over-session trends — e.g. "Improved by 3 points since last time." Gives users a longitudinal view of their mental health over time without requiring an account or login.

Future Scope — Social Media Integration

Planned

The next phase will extend the ML text analysis module to passively ingest posts from a user's own social media accounts (Twitter/X, Reddit). The same trained model will score the emotional tone of their recent writing history — tracking shifts in negative vs positive language over weeks — providing a continuous, passive risk estimation without requiring manual input. This is the full realisation of the "depression detection using social media" vision.

Results & Analysis

Overall Accuracy 93.9%
Precision 95%
Recall 93%
Training Samples 267,782

Evaluated on a held-out test set of 53,557 posts (20% stratified split). Baseline keyword model scored 78.6% — the trained model improves this by 15.3 percentage points.
